Getting Started with Mia

Loading Images

On the right side of the screen you will see a box entitled "Image I/O" which stands for Image Input and Output. You will first select an image in the file selector by double clicking on it. You can confirm it is loaded by seeing the image name in the "Image: " box below the file selector. Once you have confirmed it is selected select "Load Canvas" and you will see the image loaded!

Now that the image is loaded you can choose from a large list of filters to apply to this image (you can also use the brushes we just learned about). To learn more about how each filter works please go here or reference google

If you would like to annotate these images you can choose from the Red pen brush or the Arrow brush. To use a filter just select a number and click "Apply". For a more technical look at what the numbers on the filters mean please go here or reference google.
