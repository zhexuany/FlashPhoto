Converting and Segmenting a CT Scan Using Mia

Since you are dealing with converting and segmenting stack of CT images at one time, you should use command line mode of Mia. It is simple and powerful since it can apply every filter at one time with one line of command.

1. Slightly blur stack of CT images to reduce noise.

On the bash shell line, you can simply type ./Mia -i [First CT image directory] -blur 5.0 -o [Output CT image directory] to apply the blur filter to every CT image in the directory.

EXAMPLE) ./Mia -i brainstack/ -blur 5.0 -o brainstack/
*warning: this will overwrite the images, to not overwrite use a different -o directory*

Mia will automatically apply the blur to a stack of images and save the output images in the same directory.

2. Quantize the result into 4 segments.

On the bash shell line, you can simply type ./Mia -i [First CT image directory] -quantize 4 -o [Output CT image directory] to quantize every CT image in the directory by 4.

EXAMPLE) ./Mia -i brainstack/ -quantize 4 -o brainstack/
*warning: this will overwrite the images, to not overwrite use a different -o directory*

Mia will automatically quantize every image by 4 and save the output images in a new directory.

For any more questions type "./Mia -h" without the quotes and you will get a full list of available commands.
