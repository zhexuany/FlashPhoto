Getting Started with FlashPhoto

FlashPhoto has many exciting features that you can use to fulfill your creative desires. The many tools for this creative outreach include various brushes and filters! Below you will learn how to use these cool features.

Brushes

You will first select the wanted brush from the Tool Type list. Once the tool is selected you can then draw in the large white square and it's just like using a piece of paper! This behavior can be mimicked for any of the tools in the "Tool Type" list. Note: if you ever make a mistake just hit the undo and redo buttons! You can also change the colors of these tools by selecting a color just below the "Tool Type" list.

Filters

What if you are in the mood for playing with images though? We have plenty of options for that as well. On the right side of the screen you will see a box entitled "Image I/O" which stands for Image Input and Output. You will first select an image in the file selector by double clicking on it. You can confirm it is loaded by seeing the image name in the "Image: " box below the file selector. Once you have confirmed it is selected select "Load Canvas" and you will see the image loaded!

Now that the image is loaded you can choose from a large list of filters to apply to this image (you can also use the brushes we just learned about). To use a filter just select a number and click "Apply". For a more technical look at what the numbers on the filters mean please go here or reference google.
